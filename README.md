# Drupal Event Manager Module

## Overview
Custom Drupal 10 module that provides event listing and registration functionality.

## Features
- Custom form using Drupal Form API
- Event listing page
- Form validation and submission
- Clean routing system

## Tech Stack
- Drupal 10
- PHP
- Form API

## Routes
- /events → Event listing
- /event/register → Registration form

## Setup
1. Copy module to /web/modules/custom/
2. Enable module:
   drush en event_manager -y
3. Visit /events or /event/register
